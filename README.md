# Pokedex-Website
Dark Mode Pokédex App A web-based Pokédex built with HTML, CSS, and JS, served via a simple Python HTTP server. It features a dynamic gallery fetching Pokémon stats and types from a JSON file. Users can select two Pokémon for a battle simulation determined by combined power stats. The project also includes a standalone login interface.
